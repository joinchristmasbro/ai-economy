# dados/

Drop zone pra arquivos que você quer que o sistema analise.

## Tipos aceitos

- CSV, XLSX (planilhas, exports de CRM, listas)
- TXT, MD (transcrições, notas)
- PDF (relatórios, briefings, contratos)
- PNG, JPG (screenshots, prints de Instagram, mockups)

## Como usar

1. Arrasta o arquivo pra esta pasta
2. Diz ao Claude: "analisa dados/[nome-do-arquivo]"
3. O Claude lê e gera resumo executivo + insights

## Importante

Esta pasta é gitignored — arquivos aqui NÃO são commitados pro GitHub. Útil pra dados sensíveis de cliente que você não quer subir.

Se quiser commitar algum arquivo específico, move pra outra pasta antes.
