# Cérebro do nicho

Este é o núcleo do workspace. Toda inteligência sobre o nicho que você atua vive aqui. Cada arquivo numerado cobre um bloco do Framework Central.

## Mapa de conteúdo

- [[00-perfil-cliente-alvo]] — quem é o cliente-alvo dentro do nicho
- [[01-dores]] — dores primárias e secundárias com evidência
- [[02-custo-inacao]] — o que o cliente perde se não resolver
- [[03-objecoes]] — objeções comuns que aparecem em vendas
- [[04-linguagem-tipica]] — pain language + outcome language nativos
- [[05-resultados-vendaveis]] — métricas que importam pra esse nicho
- [[06-atuadores]] — competidores e players (preenchido manualmente ou por análise)
- [[07-tendencias]] — tendências atualizadas via /escanear-concorrencia
- [[08-benchmarks]] — show-up, close, CPL, CAC, LTV típicos
- [[09-tamanho-mercado]] — quantas empresas, crescimento, distribuição
- [[10-headlines]] — 3 ganchos de headline derivados do nicho
- [[11-riscos-compliance]] — guardrails e restrições do segmento

## Como usar

- `/setup` popula esses arquivos com pesquisa inicial
- `/mapear-mercado` aprofunda cada bloco
- `/escanear-concorrencia` atualiza [[07-tendencias]] periodicamente
- `/atualizar` mantém os arquivos sincronizados quando aprende algo novo
